We can use CSS function "calc" and ask it nicely to measure the remaining height - and mix the units used as parameters

            .side-content {
                height: ~"calc(100% - 100px)";
                position: fixed;
                top: 100px;
            }

Extra ~"" signs are added to stop processing in LESS
