Just like in one of the previous examples, we could use the "Calc" function and mix units

This time our friend is the "vw" unit - 100vw equals the width of the viewport

So we are inside the 10/12 part of an element positioned in the middle of the screen...

            .tab-header {
                width: ~"calc((100vw - 120%) / 2 + 100%)";
            }
