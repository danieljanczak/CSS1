Problem 7: Select the first of several elements with the same class

  • CSS allows to select the first child of any element, the first element of a type...
  • ... but not the first element with a specific class
  • Similar problem applies to the last elements

Solution:
  • You can select all elements of a class - and all elements of a class followed by another element of the same class, directly or anywhere else on the same level
