Problem: we would like to create a series of overrides for Bootstrap

  • In current versions of Bootstrap each grid is divided into 12 columns
  • If you nest grids, you have 12 columns inside any other width of column
  • That's perfectly OK unless you want elements in a few nesting levels to have the same sizes

Solution:
  • You can create overrides for default Bootstrap classes
  • Creating them manually would be a pain - let's see how LESS or SASS can help us
