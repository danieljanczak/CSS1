The solution - create a series of overrides to defined how many columns you want inside the grid

Doing it manually is of course possible, but even if you override just col-xs and col-xs-offset that's lots of work

Using LESS it's much easier

@grid-column-override-count: 11;
.grid-column-override-func (@grid-column-override-count) when (@grid-column-override-count > 1) {
    .grid-col-override-@{grid-column-override-count} {
        .col-xs-override(@cols) when (@cols > 0){
            .col-xs-@{cols} {
                width: (@cols/@grid-column-override-count*100%);
            }
            .col-xs-offset-@{cols} {
                margin-left: (@cols/@grid-column-override-count*100%);
            }
            .col-xs-override(@cols - 1);
        }
        .col-xs-override(@grid-column-override-count);
    }
    .grid-column-override-func(@grid-column-override-count - 1);
}
.grid-column-override-func(@grid-column-override-count);
