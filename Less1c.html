As a result we get a set of overrides that specify how the grid should be divided

            .grid-col-override-5 .col-xs-5 {
                width: 100%;
            }
            .grid-col-override-5 .col-xs-offset-5 {
                margin-left: 100%;
            }
            .grid-col-override-5 .col-xs-4 {
                width: 80%;
            }
            .grid-col-override-5 .col-xs-offset-4 {
                margin-left: 80%;
            }
            .grid-col-override-5 .col-xs-3 {
                width: 60%;
            }

If we use the override with the same amount of columns as the parent container then the columns inside the override will have the same widths as columns in the parent
