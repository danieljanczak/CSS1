Problem: vertical size of DIVs in Bootstrap rows

  • A very common scenario: Bootstrap row with a few columns
  • We would like elements in all columns to have the same height
  • As you may have seen in the past, height: 100% is not an answer
  • And no, we don't want to use JS to calculate element sizes

Solution:
  • Expand the padding
  • Use negative margins to leave the following elements where they are
  • Make overflowing padding invisible
