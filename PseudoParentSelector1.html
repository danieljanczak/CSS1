Problem: parent selector?

  • Sometimes you need to modify some aspect of the parent element
  • It should be easy, you could just use parent selector
  • Except that it doesn't exist in CSS3...
  • ... although supposedly will be added in CSS4

Solution:
  • There's no perfect solution, but if you know the relative sizes you can work around it
  • Create invisible sibling and select using +
  • Expand outside of current dimensions and go back.
