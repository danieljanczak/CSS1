Why this presentation?

  • After all discussions about plain JS vs jQuery vs Knockout vs Angular vs React vs others, when the dust settles down, you are always left with 2 constants
    • HTML
    • CSS

  • You don’t always have the full control over all components of your program:
    • Some 3rd party tools won’t give JS reliable access to internal events
    • But many 3rd party tools will assign CSS classes to modified elements
  • JS layout change tracking may be unreliable
  • UX Designers may give you design which is not easy to achieve in Bootstrap
  • Many features are missing in CSS, but sometimes you can get around them
  • Stockholm Syndrome?

Hopefully some of the following information will help you in the future
